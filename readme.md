# Aplikasi Manajemen Akademik (KRS & Nilai) 
Aplikasi ini merupakan sistem informasi akademik berbasis Java Desktop (Swing) yang dirancang untuk mengelola data Kartu Rencana Studi (KRS) dan penilaian mahasiswa secara efisien menggunakan database MySQL.

## Fitur Utama

- Implementasi Desain & Coding Pengolahan Data Array: Menggunakan struktur data array untuk memproses input NIM, Nilai Tugas, UTS, UAS, dan Absensi secara sistematis.
Sistem CRUD: Implementasi lengkap Create, Read, Update, dan Delete pada seluruh modul utama aplikasi.

- Kalkulasi Nilai Otomatis: Sistem secara otomatis menghitung nilai akhir berdasarkan bobot yang ditentukan dari komponen nilai yang diinput.

- Sistem Multi-Aktor (Role-Based Access Control) Aplikasi ini memiliki dua hak akses dengan fungsi yang berbeda: Admin: Mengelola (Tambah/Edit/Hapus) data User/Akun sistem. Mengelola master data Mahasiswa. Mengelola master data Dosen. Mengelola master data Mata Kuliah. Operator: Melakukan transaksi penginputan KRS (Kartu Rencana Studi). engelola input nilai akademik dan penentuan nilai akhir mahasiswa.
- Teknologi yang Digunakan Bahasa Pemrograman: Java Library GUI: Java Swing (NetBeans IDE) Database: MySQL Konektor: JDBC (Java Database Connectivity) Komponen Tambahan: JDateChooser (untuk input tanggal)
